🎬 Netflix Movies & TV Shows Data Analysis

📌 Project Overview :

This project explores and analyzes the Netflix Movies and TV Shows dataset available on Kaggle. The goal is to uncover meaningful insights about content distribution, trends over time, genre popularity, and regional production patterns on Netflix.
By applying data cleaning, exploratory data analysis (EDA), and visualization techniques, this project answers key questions about how Netflix’s content library has evolved.
 
📂 Dataset Information :

•	Source: Kaggle

•	Dataset Name: Netflix Movies and TV Shows

•	Features include:

o	Title

o	Type (Movie / TV Show)

o	Director

o	Cast

o	Country

o	Date Added

o	Release Year

o	Rating

o	Duration

o	Genres (Listed In)

o	Description

 
🎯 Objectives :

•	Understand the distribution of Movies vs TV Shows

•	Analyze content growth over the years

•	Identify top genres and categories

•	Explore country-wise content production

•	Examine ratings and duration patterns

•	Detect trends in content addition on Netflix

 
🛠️ Tech Stack :

•	Programming Language: Python 

•	Libraries Used:

o	Pandas

o	NumPy

o	Matplotlib

o	Seaborn

o	Plotly 

 
🧹 Data Cleaning Steps :

•	Handled missing values in columns like ,cast, date_added , rating,duration and country.

•	Converted date_added to datetime format

•	Standardized categorical columns

•	Split multi-value fields (e.g., genres, cast)

•	Removed duplicates (if any)


📊 Key Insights :

•	📌 Over 70% of Netflix content consists of Movies, showing a stronger focus on film-based content

•	📈 Content growth surged after 2015, with peak additions around 2019–2020

•	🌍 Top contributing countries: United States, India, United Kingdom

•	🎭 Most common genres: International Movies, Drama and Comedy

•	🔞 Majority ratings (TV-MA, TV-14) indicate a focus on mature audiences

• 🌍 The United States leads in both movies and TV shows, reinforcing its position as Netflix’s primary content hub; whereas India shows a stronger preference for movies 🎬, and The United Kingdom leans more toward TV shows 📺

• 🎯 July has the highest number of content releases, followed by December and September—indicating Netflix aligns releases with peak audience engagement periods.

•	⏱️ Typical movie duration: 90–120 minutes

 
📈 Analysis Highlights :

•	Distribution of Movies vs TV Shows

•	Year-wise content growth trends

•	Genre popularity analysis

•	Country-wise content production

•	Ratings and duration patterns

 
🧠 Key Learnings :

•	Handling real-world messy data with missing and inconsistent values

•	Extracting insights from multi-label categorical features

•	Building clear and meaningful data visualizations for storytelling

•	Understanding how data can reflect business strategy in streaming platforms

 
📌 Conclusion :

The analysis reveals that Netflix has aggressively expanded its catalog post-2015, with a strong emphasis on movies and mature content.
It also highlights the growing importance of international content, reflecting Netflix’s global audience strategy.
